1. 系统首页

1.1 首页

首页顶部包含“应用列表”、“家庭中心”、“时间日期控制”、“天气控制”、“消息中心”、“设置”、“音量控制”、“无线网络控制”、“音乐播放控件”。 首页左侧为“轮播图”与“冰箱温控快捷操作”，右侧为“应用快捷入口”。

  • ·点击“应用列表”，展开应用列表，显示固定配置的应用和运营配置的应用。
  • ·点击“家庭中心”，进入家庭中心主页，如用户未登录显示提示登录弹窗。
  • ·点击“时间日期控制”，进入可调节当前时间。
  • ·点击“天气控制”，进入天气预报详情页面。
  • ·点击“消息”，进入消息详情页面。有未读信息时右上角显示红点。
  • ·点击“设置”，进入设置页面。
1. 系统首页
  • ·点击“音量控制”，可调节音量。
  • ·点击“无线网络控制”，进入设置页面。
  • ·点击“音乐播放控件”，暂停或播放当前音乐。
  • ·点击“轮播图”，进入后台配置的对应跳转页面。
  • ·点击“冰箱温控快捷操作”，进入冰箱温控界面。

1.2 应用列表

应用列表包含2个部分，固定配置应用与运营配置应用，其中固定配置应用包含“影音”、“食谱”、“商城”、“相册”、“留言板”、“定时器”、“食材管理”，可以拖拽至右侧查看更多信息。运营配置应用的图标和文字是后台运营配置的，显示的内容是动态的，且不可拖拽查看更多信息。

  • ·点击“影音”，进入影音页面。
  • ·点击“食谱”，进入食谱页面。
1. 系统首页
  • ·点击“商城”，进入商城页面。
  • ·点击“食材管理”，进入食材管理页面。
  • ·点击“留言板”，进入留言板页面。
  • ·点击“相册”，进入相册页面。
  • ·点击“定时器”，进入定时器页面。

1.3 应用快捷入口

点击应用列表内应用2秒后拖拽至右侧。可以查看应用的快速预览。

  • ·“影音”：显示最近热播的影音内容，点击进入影音页面。
  • ·“食谱”：显示推荐的食谱内容，点击进入食谱页面。
  • ·“商城”：显示商城的推荐商品，点击进入商城页面。
1. 系统首页
  • ·“食材管理”：显示当前冰箱内食材的状态，点击进入食材管理页面。
  • ·“相册”：显示最近的相册图片，点击进入相册页面。
  • ·“留言板”：显示最近的留言内容，点击进入留言板页面。
  • ·“定时器”：显示定时器倒计时内容，点击进入定时器页面。

1.4 应用快速预览

点住右侧应用快捷入口2秒后拖拽到左侧查看应用快速预览模式。

1.4.1影音快速预览模式

1. 系统首页

显示音乐，视频，电台标签页的推荐内容。点击跳转对应推荐内容的详情页。底部为音乐控制器。可以切换/暂停/播放音乐。

1.4.2 食谱快速预览模式

显示今日的推荐食谱。点击显示食谱详情，点击“开始”，进入食谱板块打开对应的食谱详情。

1. 系统首页

底部为当前食谱需要使用的食材，点击购买食材进入“食材采购单”页面。

1.4.3 商城快速预览模式

显示商城前四个推荐标签内的商品，点击商品进行对应的商品详情页。点击商品卡片右下角“添加至购物车”，添加当前商品到购物车，点击购物车内的商品进行购物车页面。

1.4.4 食材管理快速预览模式

显示当前冰箱内最近储存的是食材。点击添加食材进入食材板块，添加食材界面。点击其他食材图片进入食材管理板块。底部为冰箱配菜页面，显示当前冰箱内的食材可以做的菜，点击右侧的“冰箱配菜”进入食材板块，冰箱配菜页面。

1. 系统首页

1.4.5 食材管理快速预览模式

显示最近留言板的留言内容，点击留言卡片，进入对应的留言详情。点击底部的任意方式添加留言的方式，进入留言板的添加留言界面。

1.4.6 相册快速预览模式

显示最近添加到家庭相册的照片，点击后进入相册选

1. 系统首页

中当前照片显示。底部显示当前相册内的图片点击进行切换。

1.4.7 定时器快速预览模式

显示当前的定时器，点击进入定时器页面。底部为默认定时器时间，点击开启对应的定时器。

2. 登陆/引导流程

2.1 新设备引导流程

2.1.1 欢迎页

首次启动冰箱或冰箱恢复出厂设置会进入冰箱默认的欢迎页，点击“开始设置”，进入“无线网络设置”页面。

2.1.2 无线网络设置

完成网络连接点击“下一步”，进入“用户协议页面”。

2. 登陆/引导流程

2.1.3 海尔用户协议

点击“不同意”返回“无线网络设置”，点击“同意”进入“用户隐私页面”。

2.1.4 海尔隐私协议

点击“不同意”返回“无线网络设置”，点击“同意”进入登录页面。

2. 登陆/引导流程

2.1.5 登录页

登录页面，点击“上一步”返回“无线网络设置”页面，完成登录后如果未创建家庭进入创建家庭页面，如果已创建家庭则进入首页。

2.1.6 创建家庭引导页

点击“立即创建”，进入成员信息完善页。

2. 登陆/引导流程

2.1.7 完善家庭管理员信息

可以填写家庭名称，完善家庭管理员的相关信息。完善信息后，点击“下一步”进入首页。如果冰箱有遗留数据，则提示是否同步数据。

2.1.8 同步数据页

可以选择同步到当前登录的账户或删除冰箱上的数据，完成后进入首页。

2. 登陆/引导流程

2.1.9 不连接无线网络

用户可以不连接无线网络，只使用冰箱基础设置，在“无线网络设置”页面，点击右下角“冰箱基础设置”按钮进入“冰箱基础设置”页面，如下图（具体设置页面样式根据冰箱的实际型号显示会不同 ）。点击左下角“返回按钮”，返回“无线网络设置”页面。

2.2 旧设备登陆提醒

每次升级后判定用户为未登陆状态时，提示用户登陆。用户退出登陆后显示该弹窗。登陆引导弹窗关闭后直接使用其他软件功能。在退出登陆或者未登陆时，每次使用首页的软件功能会弹出登录提示框。

2. 登陆/引导流程

2.3 网络异常

断网情况在3种情况下出现：1.新设备用户已登陆断网2.旧设备用户已登陆断网3.旧设备用户未登陆断网。此时将直接弹出“断网警告”弹窗。

3. 健康中心

3.1 未创建家庭

用户未创建家庭，进入健康中心，点击任意功能进入创建家庭流程。

3.2 健康中心主页

左侧显示当前家庭成员的基本健康详细，点击进入健康中心个人页。底部为当前时段的健康食谱推荐·，点击“食谱·”进入食谱详情页，点击“查看饮食计划”，进入家庭健康食谱推荐页。

3. 健康中心

3.3 健康中心个人页

3.3.1 健康报告

左侧显示用户的基本信息与健康评价，点击底部“编辑个人身体信息”可以再次编辑用户的基本信息。右侧为用户的各项检测最近一次的检测记录，点击对应的体质指标进入体质趋势图。

3.3.2 体质趋势图

点击顶部进行体质指标切换。在趋势图上手指滑动显示对应检测时段的体质指标。点击“上周”、“下周”查看前一周与后一周的体质指标趋势图。点击右侧“历史”图标。可以快速查看某天的体质指标。

3. 健康中心

3.3.3 健康方案

显示当前成员的健康方案，点击“更换”可更换当前的健康方案。底部显示用户选择的健康标签所对应的健康文章。点击右下角“体质检测”进入体质检测流程页面。

3. 健康中心

3.3.4 变更方案页面

选择对应需要健康方案，点击返回后会重新更新成员的食谱推荐内容。

3.4 体质检测

3.4.1 选择成员

选择需要进行检测的成员头像，点击“确认”，进入检测设备选择。

3. 健康中心

3.4.2 选择设备

选择需要使用的检测设备，点击“使用”，进入设备检测（当前仅支持思麦德体脂秤）。

3.4.3 进行检测

当用户站立在体脂秤上，并点击“开始检测”，进入检测状态，检测完成后进入个人的健康报告页。

3. 健康中心

3.5 家庭健康食谱推荐

3.5.1 食谱推荐首页

显示当前家庭成员的健康饮食内容，点击右上角可以切换单个成员的健康食谱推荐，点击购买食材进入食材的购买页面。

3.5.2 购买食材

可多选对应的食材查询需要准备的食材，选中食谱后，点击“确定”，进入采购单页面。

3. 健康中心

3.6 采购单

3.6.1 采购单首页

显示所选择的食谱对应需要的食材，可根据选择自行选择需要的食材讲采购单发送到手机或进行线上购买。如果当前冰箱的食材管理有未过期的食材也会显示对应的标记。

3.6.2 发送到手机

显示用户所需要购买的食材与所选食材对应的食谱，扫描右侧的二维码，在手机上打开对应的采购单页面。

3. 健康中心

3.6.3 线上购买

显示用户所选的食材在商城中所匹配的食材，可以选择“综合排序”或“价格最低”进行商品查询。未匹配的商品显示未匹配状态。 点击数量修改可以修改需要购买食材的数量，点击“换一个商品”进入商品替换页面。 当完成食材选择，点击“加入购物车结算”，将所选商品及数量添加至购物车。

3. 健康中心

选择对应的商品，切换选择需要购买的商品，点击“确定”返回食材购买页面。

3.7 健康头条

点击首页“查看更多”，进入健康头条列表页，查看全部健康推荐文章。点击对应的标签，显示包含标签的文章。点击对应的文章进入文章详情页。

4. 家庭中心

4.1 家庭中心首页

4.1.1 未登录

点击左上角家庭中入口，显示提示登录弹窗，完成登录后进入首页，如有数据，先进行数据同步选项选择。

4. 家庭中心

4.1.2 未登录

顶部左侧为家庭信息编辑区域，点击后可以编辑家庭名称或删除家庭群组。 顶部右侧为家庭与成员切换，点击“家”显示家庭中心主页，点击“个人头像”显示家庭中心个人页。

页面左侧为家庭钱包，显示家庭管理员的账户余额信息

  • ·点击“充值”进入余额充值页面。
  • ·点击“余额明细”进入余额明细页面。
  • ·点击“优惠券”进入“我的优惠券”界面。
  • ·点击“购物卡”进入“我的购物卡”界面。

页面右侧分别为订单信息与收藏信息。

  • ·点击“我的订单”中的全部进入“我的订单”界面，点击4个状态，分别在“我的订单”

界面筛选出4个状态。点击物流信息中的订单进入对应的订单详情页。

  • ·点击“收藏”中3个版块的图标，分别进入3个版块的收藏页面。
4. 家庭中心

4.2 家庭中心个人页

4.2.1 个人页面

左侧为用户的基本信息，点击右上角，进入个人信息编辑页面。右侧为用户“健康信息”与“标签”，如果用户已经完成了体质信息的填写，则显示用户当前的健康身体评价，点“健康报告”进入对应成员的健康中心个人页。如果用户未完成体质信息的填写则提示用户完善体质信息。右侧“我的标签”点击编辑调整用户的偏好标签。

4. 家庭中心

4.2.2 编辑个人信息

点击“用户头像”进行头像编辑，点击“用户昵称”进行昵称编辑，未绑定手机号成员，点击手机号码进行手机号绑定。点击“退出登录”退出当前家庭。点击“返回”回到当前成员的个人页面。

4. 家庭中心

4.3 添加新成员

4.3.1 登录/注册/无手机号创建

用户可以使用手机号登录已有账户或创建新账户来添加成员，也可以不使用手机进行虚拟家庭成员的创建。

4.3.2 选择身份

选择对应身份完成昵称，或选择其他自由进行昵称编辑。

4. 家庭中心

4.3.3 选择头像进行创建

选择家庭中使用的头像，点击“完成”完成成员创建，点击“继续填写”，补充成员的体质信息。

4.3.4 完善个人体质信息

填写成员默认的性别、生日、身高、体重。点击“下一步”进入个人偏好标签。

4. 家庭中心

4.3.5 选择个人偏好标签

选择对应分类的偏好标签完成成员创建。进入当前成员的家庭中心个人页。

5. 冰箱设置

5.1 501机型设置页面

点击系统首页状态栏最右侧的“设置”图标，进入冰箱设置页面。
